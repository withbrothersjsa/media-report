# 주간 리포트 생성 — 사용법

## 무엇인가
매주 1회(권장: 월요일 아침) **지난 7일** 업계 동향을 수집해
**카드뉴스 형태의 HTML 리포트**를 만드는 **수동 실행** 도구입니다.
자동 예약은 걸려 있지 않습니다. 원할 때 직접 돌립니다.

## 실행 방법
1. 이 폴더(`업계동향-클로드`)에서 **`리포트생성.bat` 더블클릭**
2. 검은 창이 뜨고 Claude가 대화형으로 열립니다.
3. Claude가 `CLAUDE.md` 지시문대로 지난 7일 동향을 검색·정리하고,
   각 기사 썸네일(og:image)을 자동 추출해 카드뉴스 리포트를 만듭니다.
   진행 상황을 보면서 필요하면 승인/수정하세요.
4. 끝나면 **`리포트/2026년 N월 M주차 업계 동향 및 주요 이슈.html`** 이 생성되고, 루트 `index.html`(최신본)도 갱신됩니다.

## 결과물 확인
- `index.html`(최신본) 또는 `리포트/YYYY년 N월 M주차 업계 동향 및 주요 이슈.html` 을 더블클릭해 브라우저로 확인.
- 과거 회차는 **`리포트/` 폴더**에 주차별 파일명으로 차곡차곡 쌓입니다.
- 카드는 **이미지 포함 어디를 눌러도** 해당 기사 원문으로 이동합니다(새 탭).
- 주차 계산: 그 주(7일)가 **더 많이 걸친 달** 기준, 그 달의 몇 번째 주인지로 매김.
  (예: 6/22~28 → 6월 4주차, 6/29~7/5 → 7월 1주차 — 7월이 5일로 더 많음)

## 폴더 구조
```
업계동향-클로드/
├── 리포트생성.bat / .ps1   ← 실행 도구
├── CLAUDE.md / 사용법.md    ← 지시문 · 안내
├── index.html               ← 가장 최근 회차 사본(빠른 보기용 · 디자인 템플릿)
└── 리포트/                  ← 회차별 산출물 보관
    ├── report_2026-06-24.html               (이전 디자인 회차)
    └── 2026년 6월 4주차 업계 동향 및 주요 이슈.html
```

## 나중에 완전 자동(예약)으로 바꾸려면
Windows **작업 스케줄러**에 `리포트생성.bat` 을 매주 월요일 아침으로 등록하면
PC가 켜져 있는 한 자동 실행됩니다. (필요할 때 요청하면 등록까지 도와드립니다.)
