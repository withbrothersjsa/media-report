# 미디어커머스 업계 동향 모니터링 (작업 지시문)

## 목적
자사 4개 브랜드가 속한 업계의 동향·특이사항을 주기적으로 크롤링·정리한다.
결과물은 HTML 리포트로 만든다.

## 자사 브랜드 & 모니터링 버티컬
- 맘앤트윈스 — 화장품/뷰티 (스킨케어·색조)
- 리아네이처 — 화장품/뷰티 (스킨케어·색조)
- 리끌로우 — 아이웨어(선글라스·안경), 레인부츠·패션잡화
- 더리아 — 뷰티 디바이스(홈뷰티 기기)

## 추적 카테고리 (이 4개로 섹션을 나눈다)
1. 화장품/뷰티
2. 아이웨어(선글라스·안경) + 레인부츠·패션잡화
3. 뷰티 디바이스(홈뷰티 기기)
4. 미디어커머스 공통 — 라이브커머스, 커머스 플랫폼(쿠팡·네이버·올리브영 등) 정책, 인플루언서/퍼포먼스 마케팅 트렌드

## 수집 항목 (각 카테고리에서)
- 신제품·신규 브랜드 런칭
- 경쟁사 마케팅/캠페인 사례
- 라이브커머스·플랫폼 동향 및 정책 변화
- 규제·인증·이슈 (화장품법, KC인증, 의료기기 광고 규정 등)
- 소비 트렌드·시장 데이터

## 참고 소스 (상황에 맞게 가감)
- 국내 커머스/마케팅 매체: 플래텀, 바이라인네트워크, 아웃스탠딩, 모비인사이드, 전자신문, 디지털데일리
- 뷰티 전문지: 코스인코리아, 뷰티누리(화장품신문)
- 종합/경제: 네이버뉴스 '미디어커머스/라이브커머스' 키워드, 매일경제, 한국경제
- 해외(선택): Modern Retail, Glossy, Digiday

## 작업 절차
1. 위 카테고리별로 최신(기본: 지난 7일) 기사·소식을 검색·수집한다.
2. 중복·단순 광고성 기사를 제거하고, 자사 브랜드 관련성이 높은 순으로 정리한다.
3. 카테고리별 섹션으로 묶어 HTML 리포트를 생성한다.
4. 각 항목은 다음 형식으로 작성한다:
   - 제목
   - 핵심 요약 (1~2줄)
   - 원문 링크
   - 시사점 (가능하면 1줄, 자사 관점에서)

## 결과물 규격
- 형식: **카드뉴스 가로 스크롤 레이아웃** HTML 리포트 1개 (카테고리별 섹션, 좌우 스와이프)
- 디자인 템플릿: 루트 `index.html`(직전 회차) 구조를 복제(내용만 교체)
- 저장 위치: **`리포트/` 폴더 안** `YYYY년 N월 M주차 업계 동향 및 주요 이슈.html` + 루트 `index.html`(최신본 사본)
- 주차 계산: 그 주(7일)가 더 많이 걸친 달 기준 + 그 달의 몇 번째 주 (예: 6/29~7/5 → 7월 1주차)
- 카테고리당 카드 6~7개 목표. 각 카드 = 썸네일 이미지 + 컬러 태그 + 제목 + 요약 + 시사점 + 출처
- 카드 이미지는 기사 og:image 를 자동 추출(원본 HTML 파싱; WebFetch 금지). 한글 URL은 percent-encoding.
- 카드 전체가 클릭되어 기사로 이동(stretched-link CSS 유지)
- 리포트 상단: 생성일 + 수집 기간 + 핵심 헤드라인 5개(번호 배지). 담당 브랜드 줄은 넣지 않는다.

## 응답 규칙
- 모든 응답과 설명은 항상 한국어로 작성한다.
- PowerShell 터미널에 표시되는 모든 화면(출력, 안내 메시지, 오류 메시지 등)도 한국어로 작성한다.
- Claude가 제공하는 모든 텍스트(설명, 요약, 시사점, UI 문구 등)는 예외 없이 한국어를 사용한다.
